# Security Log Analyzer

## Overview
A simple Python-based security log analyzer that detects failed login attempts
and flags potential brute-force attacks based on a threshold.

## Features
- Parses log files
- Detects failed login attempts
- Counts attempts per IP address
- Prints security alerts

## Technologies Used
- Python
- File handling
- Dictionaries

## How to Run
```bash
python analyzer.py
